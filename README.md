Diomaps 🌍

O Diomaps é uma ferramenta tecnológica desenvolvida como Projeto de Conclusão de Curso (TCC), focada em facilitar o descarte correto de resíduos e promover a conscientização ambiental.

🚀 Sobre o Projeto

A plataforma surge para resolver a dificuldade de encontrar pontos de coleta seletiva e a falta de informação centralizada sobre reciclagem. O Diomaps combina geolocalização e educação ambiental em uma única interface.

✨ Funcionalidades Principais

Localização de Postos: Sistema de busca georreferenciada para encontrar pontos de coleta próximos ao usuário.

Cadastro de Pontos: Módulo para registro de novos centros de reciclagem e postos de descarte.

Blog Ecológico: Espaço dedicado a artigos, guias e notícias sobre o ecossistema da sustentabilidade.

Painel Administrativo: Interface para gestão de dados e moderação de novos cadastros.

🛠️ Tecnologias Utilizadas

Frontend: [HTML & CSS e JS] Backend: [PHP] Banco de Dados: [MySQL]
